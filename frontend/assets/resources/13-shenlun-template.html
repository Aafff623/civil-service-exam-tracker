一、归纳概括题

考查提炼要点、概括归纳能力，要求全面、准确、简洁。

模板一：问题概括型

结构：总括句 + 一是……二是……三是……

句式：「XX 领域/工作存在以下问题：一是主观层面，……；二是客观条件，……；三是体制机制，……。」

模板二：经验做法型

结构：「XX 的主要做法包括：」+ 分条列点

句式：「一是坚持……，通过……实现……；二是强化……，建立……机制；三是完善……，推动……落地。」

要素常用引导词示例
问题一是……问题突出一是基层治理力量不足
原因究其原因，主要是……究其原因，主要是监管缺位
做法 / 经验一是……；二是……一是完善机制；二是强化保障
影响 / 意义有利于……；有助于……有利于激发市场活力
范文 · 归纳概括

题目：概括材料中农村电商发展面临的困难。（不超过 200 字）

参考答案：
农村电商发展面临以下困难：一是基础设施薄弱，冷链物流、网络覆盖不足，配送成本高。二是人才短缺，缺乏运营、直播、电商技能专业人才。三是产品标准化低，品控难、品牌弱，同质化竞争激烈。四是资金保障不足，农户融资难，平台入驻与推广费用较高。五是政策支持待细化，补贴分散、监管滞后，市场秩序有待规范。
字数控制：作答字数达到要求上限的 90% 左右，留有余地便于卷面整洁。

二、综合分析题

包括解释型、评价型、启示型等，需分析透彻、逻辑清晰。

模板一：解释型（谈谈对……的理解）

结构：释义 → 分析（原因 / 影响 / 关系）→ 总结或升华

句式：「这句话深刻揭示了……的内在关系。释义：……分析：一方面……另一方面……总结：因此，……」

模板二：评价型（谈谈看法 / 评价）

结构：表态 → 理由分析（正反）→ 对策或总结

句式：「对于……，应当辩证看待。积极意义：……存在问题：……对策建议：……」

范文 · 综合分析

题目：谈谈你对「新质生产力本身就是绿色生产力」的理解。（不超过 300 字）

参考答案：
这句话揭示了科技创新与绿色发展的内在统一关系。
释义：新质生产力以创新为主导，本身就具有高效、低碳、可持续特征，与绿色生产力理念高度契合。
分析：一方面，绿色技术突破（新能源、节能环保）本身就是新质生产力的重要组成；另一方面，新质生产力通过产业升级减少资源消耗和污染排放，推动发展方式绿色转型。二者相互促进、相辅相成。
总结：因此，培育新质生产力必须坚持生态优先，以绿色科技创新引领高质量发展。

三、贯彻执行题

应用文写作，格式分 标题、主送机关、正文、落款 四部分（按文种调整）。

模板一：倡议书 / 宣传稿

结构：标题 + 称谓 + 背景目的 + 具体倡议（分条）+ 号召结尾 + 落款日期

句式：「为……，我们向……发出如下倡议：一、……二、……让我们携手……」

模板二：汇报提纲 / 调研报告

结构：标题 + 称谓 + 基本情况 + 主要做法 + 存在问题 + 下一步建议

句式：「一、基本情况。……二、主要做法。……三、存在问题。……四、对策建议。……」

文种格式要点正文结构
倡议书标题、称谓、落款+日期背景目的 → 具体倡议 → 号召结尾
宣传稿标题、称谓（可选）引言 → 主体内容 → 呼吁总结
短评标题即可亮明观点 → 多角度论证 → 简要升华
汇报 / 提纲标题、称谓基本情况 → 主要做法 → 问题建议
通知标题、主送机关、落款缘由 → 事项 → 要求

四、提出对策题

结构：针对材料问题一一对应提出对策

句式：「一是针对……问题，……；二是针对……问题，……」

句式：「政府层面，……；企业层面，……；社会层面，……；个人层面，……」

对策表述公式

主体 + 手段 + 目的 / 对象

五、大作文（3 主题范文）

五段式结构

  1. 标题：点明主题，宜对仗或比喻
  2. 开头（150 字）：引材料 + 点题 + 亮观点
  3. 分论点 1–3（各 200–250 字）：分论点句 + 理论论证 + 事例论证
  4. 结尾（100 字）：回扣主题 + 升华展望
范文一 · 基层治理

题目：围绕「基层治理」写一篇文章。（1000–1200 字）

标题：夯实基层之基 筑牢治理之本

开头：治国安邦，重在基层。习近平总书记强调，基层强则国家强，基层安则天下安。当前，基层治理面临新老矛盾交织、群众诉求多元等挑战，唯有坚持党建引领、科技赋能、群众参与，方能打通治理「最后一公里」，夯实中国式现代化根基。

分论点一：坚持党建引领，把准基层治理「方向盘」。基层党组织是战斗堡垒。浙江「枫桥经验」历久弥新，关键在于坚持党的领导，将矛盾化解在基层。应推动资源、服务、管理下沉，发挥党员先锋模范作用，以红色引擎驱动基层善治。

分论点二：坚持科技赋能，提升基层治理「高效能」。「城市大脑」「网格化管理」等数字化手段，让「民有所呼、我有所应」更加精准。应打破数据壁垒，建设一体化治理平台，让数据多跑路、群众少跑腿。

分论点三：坚持共建共享，凝聚基层治理「向心力」。基层治理不是政府「独角戏」，而是社会「大合唱」。应完善自治、法治、德治相结合机制，拓宽群众参与渠道，形成人人有责、人人尽责的治理共同体。

结尾：基层治理是国家治理的基石。唯有夯实基层、守正创新，才能让社会既充满活力又和谐有序，为推进中国式现代化奠定坚实基础。
范文二 · 新质生产力

题目：围绕「发展新质生产力」写一篇文章。

标题：以新质生产力塑造高质量发展新优势

开头：发展新质生产力是推动高质量发展的内在要求和重要着力点。面对新一轮科技革命和产业变革，我们必须以科技创新引领产业创新，加快培育新质生产力，为现代化建设注入强劲动能。

分论点一：以科技创新为核心驱动力。新质生产力特点是创新。应强化企业创新主体地位，加大基础研究投入，突破关键核心技术「卡脖子」难题，让原创性、颠覆性科技成果竞相涌现。

分论点二：以产业升级为主攻方向。新质生产力关键在质优。应推动传统产业改造提升，培育壮大战略性新兴产业，布局建设未来产业，促进产业链向中高端迈进。

分论点三：以体制机制为保障支撑。新质生产力本质是先进生产力。应深化科技体制改革，完善人才培养、评价、激励机制，营造鼓励创新、宽容失败的社会氛围。

结尾：新质生产力既是发展命题，更是改革命题。唯有坚持创新引领、系统推进，才能在新征程上赢得优势、赢得未来。
范文三 · 文化自信

题目：围绕「文化自信」写一篇文章。

标题：坚定文化自信 汇聚精神力量

开头：文化是一个国家、一个民族的灵魂。文化自信是更基础、更广泛、更深厚的自信。推进中国式现代化，必须坚定文化自信，推动中华优秀传统文化创造性转化、创新性发展，建设中华民族现代文明。

分论点一：守护文化根脉，赓续历史文脉。中华文明绵延五千年，是自信的底气所在。应加强文物保护利用和文化遗产保护传承，让收藏在博物馆里的文物、陈列在广阔大地上的遗产、书写在古籍里的文字都活起来。

分论点二：推动文化创新，满足人民需求。文化的生命力在于创新创造。应深化文化体制改革，繁荣发展文化事业和文化产业，推出更多无愧于时代的精品力作，丰富人民精神世界。

分论点三：促进文明互鉴，提升国际影响。开放包容是文明发展的活力源泉。应讲好中国故事，传播好中国声音，展现可信、可爱、可敬的中国形象，增强中华文化国际影响力。

结尾：文化自信不是自封的，而是在传承创新中铸就、在交流互鉴中彰显的。坚定文化自信，方能凝聚起实现中华民族伟大复兴的强大精神力量。

大作文审题立意方法

  1. 抓题干关键词：圈出核心概念与限定词（如「基层」「创新」「绿色」）
  2. 回材料找立场：材料中政策导向、案例倾向即立意方向
  3. 确定总论点：一句话概括全文主张，放在开头段末
  4. 拆分分论点：从「是什么—为什么—怎么办」或「三个维度」展开
  5. 匹配论据：每段 1 个材料事例 + 1 个积累事例

综合分析题精练范文二

范文 · 综合分析（评价型）

题目：谈谈你对「数字鸿沟」的看法。（不超过 300 字）

参考答案：
「数字鸿沟」是信息化进程中不同群体在接入、使用和信息获益上的差距，应当正视并积极应对。
积极面：数字技术为政务服务、乡村振兴提供了新机遇，「互联网+」显著提升了治理效能。
问题面：老年人、偏远地区居民在数字技能上存在明显短板，「数字形式主义」也加重了基层负担。
对策：应推进数字基础设施均衡覆盖，保留线下服务渠道，开展适老化改造与数字技能培训，让技术发展成果惠及全体人民。

贯彻执行题精练范文二

范文 · 调研报告提纲

题目：撰写一份关于「社区养老服务」的调研报告提纲。（不超过 500 字）

标题：关于社区养老服务情况的调研报告（提纲）
正文：一、调研背景与目的。应对人口老龄化，了解社区养老服务现状与需求。二、基本情况。社区老年人口比例、现有养老设施与服务供给。三、主要做法。日间照料、助餐助医、志愿服务等。四、存在问题。设施不足、专业人才短缺、资金保障有限。五、对策建议。加大投入、引进社会力量、培养专业人才、推动医养结合。

提出对策题精练范文二

范文 · 提出对策

题目：针对材料中「青年就业难」问题提出对策建议。（不超过 400 字）

参考答案：
一是拓宽就业渠道，支持新兴产业、现代服务业吸纳就业，鼓励高校毕业生到基层和中小微企业就业。二是强化技能培训，开展校企合作、订单式培养，提升毕业生与市场需求的匹配度。三是完善政策支持，落实稳岗返还、创业担保贷款等惠企政策，激励企业扩大招聘。四是优化就业服务，搭建线上线下招聘平台，提供职业规划指导和心理咨询。五是规范市场秩序，打击虚假招聘、就业歧视，维护劳动者合法权益。

申论卷面与书写规范

  • 使用黑色签字笔，不得在答题卡做标记
  • 分条用「一、二、三」或「（一）（二）（三）」，条理清晰
  • 贯彻执行题格式分不可丢：标题、称谓、落款按文种要求书写
  • 大作文标题居中，分段明确，每段首行空两格
  • 字数控制在上限 90% 左右，避免大幅超出或不足
  • 答题卡每格一字，标点占一格；字迹不清视为错字
