公务员考试学习跟踪系统 — 各功能模块入口。

公务员考试学习跟踪系统

Flask + SQLite 后端与原生前端，覆盖考试资源、学习计划、个性化推荐、学习进度统计、题库练习与答疑留言等模块，适合课程设计演示与答辩。

01

学习总览

今日任务、学习时长、完成率、弱项提醒和考试倒计时。
02

考试与资源

考试大纲、历年试题、备考资料与政策公告。
03

学习计划

根据目标与时间生成可执行的学习计划。
04

学习推荐

基于薄弱项推荐资料与练习题。
05

学习统计

学习进度、正确率与连续打卡。
06

题库与练习

题目练习、解析与留言答疑。
启动：后端 python app.py（5001）+ 前端 python -m http.server 8080，登录后使用各模块。

数据来自后端 API；考试时间线为演示参考日期。
