# 三周功能完成清单

> 项目名称：公务员考试学习跟踪系统  
> 技术栈：Flask + MySQL + 原生 HTML/CSS/JS  
> 目标：完成一个可本地运行、可演示、可答辩的公务员考试学习跟踪系统。

---

## Week 1：系统基础与数据准备

### 主要任务

- 完成项目初始化与目录结构搭建
- 搭建 Flask 后端基础结构
- 搭建前端静态页面基础结构
- 设计并初始化 MySQL 数据库
- 实现用户注册、登录、登出功能
- 实现考试资源列表与详情功能
- 准备基础演示数据

### 完成内容

- 创建 `backend/`、`frontend/`、`docs/` 等项目目录
- 完成 Flask 应用入口、配置文件、数据库连接
- 设计用户、科目、资源、题目、答题记录、学习计划等核心数据表
- 编写数据库初始化脚本，支持一键导入演示数据
- 完成 Session 登录态管理
- 完成资源库页面与资源详情页面
- 导入 7 个考试科目和 23 条备考资源

### 阶段交付物

- 后端基础框架
- 前端基础页面
- MySQL 数据库结构
- 用户账户模块
- 考试资源管理模块
- 项目 README / PRD / Roadmap 初稿

---

## Week 2：核心业务功能开发

### 主要任务

- 实现题库练习功能
- 实现答题记录与判分逻辑
- 实现智能学习计划生成
- 实现学习进度统计
- 实现弱项识别
- 实现个性化推荐
- 实现题目解析与答疑留言功能

### 完成内容

- 支持单选题、多选题、判断题练习
- 支持按科目、题型、资源来源筛选题目
- 提交答案后记录答题结果并显示解析
- 根据学习目标生成每日学习计划
- 支持学习任务完成状态更新
- 统计学习完成率、答题数量、正确率等数据
- 根据答题情况识别薄弱科目
- 根据弱项推荐相关学习资源和同类题目
- 完成答疑留言接口与页面展示

### 阶段交付物

- 题库与练习模块
- 智能学习计划模块
- 学习进度统计模块
- 弱项识别与推荐模块
- 题目解析与答疑模块
- Dashboard 数据接入

## 推荐演示流程

```text
登录系统 → 查看 Dashboard → 浏览考试资源 → 进入资源详情
→ 开始题库练习 → 提交答案 → 查看统计结果
→ 查看弱项推荐 → 查看学习计划
```
