# Issue 02 — Frontend resource library page

Status: closed

## Goal

Create a resource library page where users can browse and view resources.

## Acceptance criteria

- [ ] Create `resources.html` page
- [ ] Page shows a list of resources with title, type, subject
- [ ] Users can filter by subject and type
- [ ] Clicking a resource shows its details (modal or detail view)
- [ ] Page requires login (redirect to login if not authenticated)

## Dependencies

- Issue 01 backend resource API
- Auth module for login check

## Out of scope

- Admin resource creation UI
- File upload UI
